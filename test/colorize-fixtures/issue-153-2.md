# MkDocs / Python-Markdown

The spaces are optional

```.sh
# Shell comment
```

```{.sh}
# Shell comment
```

```   {.sh}
# Shell comment
```

```{   .sh}
# Shell comment
```

```{.sh   }
# Shell comment
```

Spaces are also optional when attributes are defined

```{.sh   .class}
# Shell comment
```

```{.sh .class   }
# Shell comment
```
